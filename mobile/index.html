Help me

This application is designed to run on a majority of machines (tablets or smartphones). My website shows the machines on which the game has been validated. If you test this application on a machine not listed, I would appreciate if you leave me a message telling me on which machine you tested the application.
In case the game does not work properly, you can also leave me a message telling me your machine and screen resolution so that I improve the application.

Do not hesitate to make a donation. This would allow me to finance my work to offer you other free games and applications.
